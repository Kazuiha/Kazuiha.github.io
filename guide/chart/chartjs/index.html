chart.js

约 240 字小于 1 分钟

2024-03-16

概述

主题支持在 文章中 嵌入由 chart.js 图表。

该功能由 @vuepress/plugin-markdown-chart 提供支持。

配置

主题默认不启用该功能。

你需要在你的项目中安装 chart.js 库。

pnpm
pnpm add chart.js
yarn
yarn add chart.js
npm
npm install chart.js

然后在 .vuepress/config.ts 配置文件中，启用该功能：

.vuepress/config.ts
export default defineUserConfig({
  theme: plumeTheme({
    markdown: {
      chartjs: true, 
    },
  })
})

语法

::: chartjs 标题
```json
{
  // 此处为图表配置
}
```
:::

图标配置请查看 chart.js 文档 。

示例 Fork 自 @vuepress/plugin-markdown-chart, 遵循 MIT 许可证。
